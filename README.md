# Odds Converter Streamlit App
This is the repo for my sports betting odds conversion web app built with Streamlit.

![Odds Converter App](https://raw.githubusercontent.com/steodose/odds-converter/master/app_screenshot.png)

## Introduction
The Odds Converter Streamlit App is a versatile tool designed for sports betting enthusiasts. It allows users to easily convert sports betting odds among various international formats, including American, Decimal, and Fractional odds. The app also calculates and displays the implied probabilities associated with the given odds.

## Features
- **Odds Conversion**: Converts betting odds between American, Decimal, and Fractional formats.
- **Implied Probability Calculation**: Calculates the implied probability of a bet winning based on the entered odds.
- **Interactive UI**: Offers a user-friendly sidebar for input and displays results as easy-to-read metric cards.
- **Probability Chart**: Visualizes the win-loss probabilities in an interactive bar chart.
- **Glossary and About Tabs**: Provides additional information about betting terms and the app's purpose.

## How to Use
1. **Select Odds Type**: Choose between American or Decimal odds using the radio button in the sidebar.
2. **Enter Odds**: Input the odds in the selected format.
3. **View Results**: The app automatically converts the odds into other formats and displays the implied probability. These results are shown in the main area as metric cards.
4. **Probability Chart**: Navigate to the 'Probability Chart' tab to see the visual representation of win-loss chances.
5. **Additional Resources**: The 'Glossary' and 'About' tabs offer further information on betting terms and app details.

## Technologies Used
- Streamlit for UI
- Pandas for data manipulation
- Altair for data visualization
- Seaborn for advanced color palettes

## Setup and Installation
1. Ensure Python is installed on your system.
2. Install Streamlit using pip: `pip install streamlit`
3. Clone or download the app repository from GitHub.
4. Run the app using the command: `streamlit run odds_converter.py`
